# IRISConfigurationDiagrams

The idea comes from https://ideas.intersystems.com/ideas/DPI-I-807, implementing chart display for mirror status and the mapping status of globals, routines, and data packages.

# Install

## 1. Docker Installation
Pull the code and run docker compose up -d
## 2. Zpm Installation
Zpm "install 'irisconfigurationdiagrams'"

# USE
After the installation is complete, visit http://ip:52773/csp/user/index.csp
